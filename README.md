## Instructions for using git
### Making a new branch
```
git checkout main
git checkout -b "<name of branch>"
```

### Making a commit
```
git status
git add .
git commit -m "<commit message>"
```

### Making a pull-request
```
git push
```
Then click on link to repo to make pull request.

### Getting the latest code on main
```
git checkout main
git pull
```
